# XL-GUI - Community Maintained PySimpleGUI Fork

**XL-GUI 是 PySimpleGUI 4.60.5 的社区维护版本** - 在官方停止支持后，我们继续提供完全免费、开源的 GUI 库解决方案。

## 🎯 项目背景

> 官方既然停止支持，我就把之前的版本发布，让人人都可以用上

由于 PySimpleGUI 官方已转向商业许可模式，我们 fork 了最后一个完全开源的 4.60.5 版本，确保开发者能够继续免费使用这个优秀的 GUI 库。

## ✨ 特性优势

- ✅ **完全免费** - 无需注册、无需许可证、无任何限制
- ✅ **稳定可靠** - 基于经过验证的 PySimpleGUI 4.60.5 版本
- ✅ **无缝迁移** - 现有代码无需任何修改
- ✅ **社区驱动** - 由开发者社区共同维护和改进
- ✅ **完美兼容** - 避开了 5.x 版本的兼容性问题
